## WAD Tool

### Description
Packer/unpacker for Neversoft's "Big Guns" engine games.

* Apocalypse
* Spider-Man
* Spider-Man 2
* Tony Hawk's Pro Skater
* Tony Hawk's Pro Skater 2
* Tony Hawk's Pro Skater 3
* Tony Hawk's Pro Skater 4
